    PeerAccomplish

    A site that encouraging users to share their goals to feel motivated and get more done.

    • Deployed on Heroku
    • Automated email messages on sign up and depending on user activity using Mailgun
    • Many-to-many, one-to-many, has-many through model assocations
    • Automated scheduled tasks using Heroku Scheduler and Rake tasks
    • Omniauth authentication using omniauth-facebook with Facebook Graph API
    • Redirect for new accounts to how to use explanation
    • Backups using pg:backups
    • CSS styling using Bootstrap
    • Authentication and authorization using Devise
    • Image uploads and modification using CarrierWave and Cloudinary
